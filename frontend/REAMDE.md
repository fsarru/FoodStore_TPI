# Food Store - Frontend Web

Esta es la capa de interfaz de usuario para el sistema Food Store.

## Tecnologías Utilizadas
- TypeScript
- Vite
- HTML5 / CSS3 Puro (sin frameworks)
- LocalStorage para simulaciones de persistencia temporal

## Cómo ejecutar el proyecto
1. Abre una terminal en esta carpeta (`frontend/`).
2. Ejecuta `npm install` para descargar las dependencias.
3. Ejecuta `npm run dev` para iniciar el servidor de desarrollo Vite.
4. Abre la URL (generalmente http://localhost:5173) en tu navegador web.

## Notas sobre la implementación
- Los datos son servidos desde los archivos `.json` ubicados en `public/data/`.
- El registro de usuarios funciona y se mockea en `localStorage`.
- Los paneles de administrador incluyen funcionalidad CRUD.
